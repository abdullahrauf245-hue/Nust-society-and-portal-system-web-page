1. Magnitude Comparator (4-Bit)

A magnitude comparator identifies if A > B, A < B, or A = B[cite: 11, 12].

  • Equality Algorithm: Two numbers are equal if all bit pairs (Ai, Bi) are equal[cite: 26, 27]. We use XNOR: xi = AiBi + Ai'Bi'[cite: 28, 29].
  • Final Equality: (A=B) = x3·x2·x1·x0[cite: 32].

Logic Expressions for Inequality:

(A>B) = A3B3' + x3A2B2' + x3x2A1B1' + x3x2x1A0B0' [cite: 42]
(A<B) = A3'B3 + x3A2'B2 + x3x2A1'B1 + x3x2x1A0'B0 [cite: 43]
Sample Question

If A = 1101 and B = 1110, determine the output of the 4-bit comparator.

Solution: Check MSB down. A3=B3 (1=1), A2=B2 (1=1). At 2nd position, A1=0 and B1=1[cite: 40]. Therefore, A < B[cite: 40].

Practice

Draw the logic gate required to produce the 'x3' signal. (Answer: XNOR gate with inputs A3 and B3 [cite: 28, 29]).

2. Decoders & Demultiplexers IC 74LS138

Decoder: Converts n inputs into 2n outputs. Only 1 output is active[cite: 68].

Demultiplexer: Routes 1 data input to one of 2n outputs using selection lines[cite: 148, 149].

  • Key Fact: A decoder with an "Enable" input acts as a Demultiplexer[cite: 152].
  • Active-LOW: NAND decoders (like IC 74LS138) produce a '0' for the selected minterm[cite: 96, 117].
Sample Question

Implement F(A,B,C) = Σ(1, 2, 4, 7) using a 3x8 decoder.

Solution: Connect inputs A, B, C to decoder inputs. Connect outputs D1, D2, D4, and D7 to an external 4-input OR gate[cite: 84, 85].

Practice

A BCD-to-Seven-Segment decoder (IC 74LS47) has how many inputs and outputs? (Answer: 4 BCD inputs, 7 segment outputs [cite: 126, 127]).

3. Encoders & Priority Encoders

Performs the inverse of a decoder (2n inputs → n outputs)[cite: 159, 160].

  • Standard Encoder Limitation: Only one input can be active at a time[cite: 166, 176].
  • Priority Encoder: If multiple inputs are '1', the highest priority input is encoded[cite: 182, 188].
  • Valid Output (V): Indicates if any input is '1'. If all are '0', V=0[cite: 184, 191].

Sample Question

In a 4-input priority encoder, if D2=1 and D3=0, what are the outputs x and y?

Solution: Since D2 is active and higher than D0/D1, the binary code for '2' is produced [cite: 188]: x=1, y=0[cite: 195].

Practice

Why do we use "Don't Cares" (X) in the priority encoder truth table? (Answer: To simplify logic/condense table [cite: 194]).

4. Multiplexers (Data Selectors)

Selects 1 of 2n inputs based on n selection lines[cite: 217, 218].

Function Implementation Rule: A 2n-1-to-1 MUX can implement any n-variable function[cite: 251, 252].

Sample Question

How do you implement a 3-variable function F(x, y, z) using a 4-to-1 MUX?

Solution: Use x and y as selection lines S1, S0. The data inputs (I0-I3) will be either z, z', 0, or 1 based on the truth table pairs[cite: 256].

Practice

List two applications of MUX. (Answer: Parallel-to-Serial conversion, Signal routing [cite: 272, 273]).
